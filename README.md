Maxel
=====

Managed download accelerator similar to axel

Usage
=====
```
Usage: Maxel.exe <uri>
Maxel 1.0.0.0
Copyright (C)  2013
  -c INT, --chunksize=INT             (Default: 1048576) Size of chunks to split the file into
  -i, --ignorecerterrors              (Default: False) Ignore ssl certificate errors
  -n INT                              (Default: 10) Number of concurrent connections
  -o FILE, --output=FILE              Output file
  -p PASSWORD, --password=PASSWORD    Password
  -u USERNAME, --username=USERNAME    Username
  --help                              Display this help screen.
```
